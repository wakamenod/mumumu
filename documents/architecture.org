* 4. 設計上の注意点

** 4.1 ElapsedTimer の自己完結設計
タイマーロジックはコンポーネント内に閉じ込める。親から =startedAt= を渡すだけ。
=setInterval= + =useState= だと親の再レンダリングが秒間100回発生するため、
=requestAnimationFrame= 駆動 + =AppState= 監視の現設計を維持すること。

** 4.2 LaTeX 描画
=react-native-mathjax-html-to-svg= を使用（WebView 不使用・New Architecture 対応）。
=mathjax-full= は Metro バンドラーと非互換のため不採用。

** 4.3 iOS ビルド（Podfile パッチ）
=react-native-svg= の依存する =fmt= ライブラリが Apple Clang 16+ でビルドエラーになるため、
=ios/Podfile= の =post_install= で =base.h= をパッチしている。
=pod install= のたびに自動適用される。=ios/Podfile= は git 管理対象。

** 4.4 AdMob バージョン制約
Expo SDK 54 / RN 0.81 では =react-native-google-mobile-ads= v16 系に
config plugin 互換性問題があるため v15.8.x を使用する。
